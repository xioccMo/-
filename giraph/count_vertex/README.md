## 题目描述
输入中给出图的相关数据，要求输出图中的顶点个数

## 样例

下面将依次给出图的输入格式以及结果输出格式：

- 输入格式

    ```
    1	1	3
    2	1	3
    3	1	1	2	4
    4	1	3
    5	1	6
    6	1	5	7
    7	1	6
    8	1
    ```
    
    其中第一列表示顶点 ID，第二列表示顶点的值，其余列表示和第一列顶点之间存在边的顶点 ID
    
- 输出格式

    ```
    1,8
    2,8
    3,8
    4,8
    5,8
    6,8
    7,8
    8,8
    
    ```

    其中第一列输出顶点 ID，第二列表示图中顶点的个数
